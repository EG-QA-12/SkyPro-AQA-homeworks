## Примеры ответов

### Ответы на вопросы регистрации
```
Для регистрации ООО в Беларуси необходимо подготовить следующие документы:
1. Заявление по установленной форме
2. Устав организации
3. Протокол общего собрания учредителей
4. Документы, подтверждающие оплату государственной пошлины

Процедура занимает 3 рабочих дня с момента подачи документов.

Для регистрации ИП в Беларуси необходимо:
1. Подать заявление в орган регистрации
2. Представить документы, удостоверяющие личность
3. Оплатить государственную пошлину
4. Выбрать систему налогообложения

Срок регистрации ИП составляет 5 рабочих дней.
```

### Ответы на вопросы налогов
```
ОО в Беларуси обязано уплачивать следующие налоги:
- Налог на прибыль (18%)
- НДС (20%)
- Подоходный налог с работников
- Социальные отчисления

Для малых предприятий предусмотрены льготы по налогам:
- Упрощенная система налогообложения
- Сниженные ставки налогов
- Отсрочка по уплате налогов

ИП в Беларуси платит:
- Единый налог (при упрощенной системе налогообложения)
- Налог на прибыль (при общей системе)
- НДС (при превышении установленного порога)
- Подоходный налог с работников
```

### Ответы на вопросы договоров
```
Для получения лицензии необходимо:
1. Подать заявление в уполномоченный орган
2. Представить пакет документов
3. Оплатить государственную пошлину
4. Пройти проверку соответствия требованиям

Срок рассмотрения заявления составляет 30 календарных дней.

Для заключения договора поставки товаров необходимо:
1. Определить предмет договора
2. Установить сроки поставки
3. Согласовать цену и порядок расчетов
4. Прописать условия поставки и ответственность сторон
5. Подписать договор в установленном порядке
```

### Ответы на вопросы процедур
```
Процедура реорганизации юридического лица включает:
1. Принятие решения о реорганизации
2. Подготовку разделительного баланса
3. Уведомление кредиторов
4. Государственную регистрацию
5. Внесение изменений в ЕГР

Срок реорганизации зависит от ее формы и может занять от 2 до 6 месяцев.

Для получения разрешения на строительство необходимо:
1. Получить технические условия
2. Разработать проектную документацию
3. Пройти экспертизу проекта
4. Подать заявление в уполномоченный орган
5. Получить разрешение на строительство

Срок получения разрешения составляет 15 рабочих дней.
```

## Тестовые сценарии

### Сценарий 1: Создание и публикация вопроса
1. Создать вопрос: "Как зарегистрировать ООО в Беларуси?"
2. Найти вопрос в панели модерации
3. Опубликовать вопрос
4. Проверить что вопрос отображается на сайте

### Сценарий 2: Ответ на вопрос
1. Создать вопрос: "Какие налоги платит ИП?"
2. Найти вопрос в панели модерации
3. Ответить на вопрос: "ИП платит налог на прибыль 18%"

1. Создать несколько вопросов
2. Ответить на них с разными типами ответов

## Частые ошибки

### Ошибки при создании вопросов
- Пустой текст вопроса
- Слишком длинный текст вопроса
- Некорректная категория
- Отсутствие подкатегории для определенных категорий

### Ошибки при ответах
- Пустой текст ответа
- Ответ на несуществующий вопрос
- Некорректный тип ответа
- Нарушение форматирования текста ответа

### Ошибки публикации
- Публикация уже опубликованного вопроса
- Публикация вопроса с неправильным статусом
- Публикация ответа без модерации
- Нарушение требований к содержанию ответа

## Типы ответов

### ANSWER (Стандартный ответ)
Обычный ответ на вопрос пользователя с подробным разъяснением.

### SUPPORTIVE (Поддерживающий ответ)
Ответ, который поддерживает позицию задавшего вопрос, с дополнительными пояснениями.

### MAXIMUM (Максимально полный ответ)
Наиболее полный и развернутый ответ с дополнительной информацией и ссылками на нормативные акты.

### OTHER (Другой тип ответа)
Специальный тип ответа для особых случаев, не подпадающих под другие категории.

## Статусы вопросов и ответов

### Статусы вопросов
- pending - вопрос ожидает модерации
- published - вопрос опубликован
- rejected - вопрос отклонен
- archived - вопрос архивирован

### Статусы ответов
- draft - черновик ответа
- pending - ответ ожидает модерации
- published - ответ опубликован
- rejected - ответ отклонен
