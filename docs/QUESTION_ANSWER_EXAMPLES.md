# Примеры вопросов и ответов для тестирования

## Введение

Этот документ содержит примеры вопросов и ответов, которые используются 
в тестах для проверки функциональности системы вопросов и ответов.

## Примеры вопросов

### Вопросы о регистрации
```
Как зарегистрировать ООО в Беларуси? Какие документы нужны?
Как зарегистрировать ИП в Беларуси? Сколько времени занимает процедура?
Какие существуют виды договоров займа в Беларуси? Каков порядок получения?
Процедура ликвидации юридического лица в Беларуси. Какие документы требуются?
```

### Вопросы о налогах
```
Какие налоги платит ООО в Беларуси? Какие ставки применяются?
Какие налоги платит ИП в Беларуси? Какие льготы предусмотрены?
Какие сборы и платежи обязательны для предприятий в Беларуси?
```

### Вопросы о договорах
```
Какие существуют виды лицензий в Беларуси? Какие требования предъявляются?
Какие существуют виды разрешений в Беларуси? Какие документы нужны?
Какие существуют виды соглашений в Беларуси? Сколько стоит оформление?
```

### Вопросы о процедурах
```
Процедура реорганизации юридического лица в Беларуси. Какие особенности?
Процедура перерегистрации предприятия в Беларуси. Какие требования предъявляются?
Процедура изменения юридического лица в Беларуси. Какой порядок действий?
```

## Примеры ответов

### Ответы на вопросы регистрации
```
Для регистрации ООО в Беларуси необходимо подготовить следующие документы:
1. Заявление по установленной форме
2. Устав организации
3. Протокол общего собрания учредителей
4. Документы, подтверждающие оплату государственной пошлины

Процедура занимает 3 рабочих дня с момента подачи документов.
```

### Ответы на вопросы налогов
```
ООО в Беларуси обязано уплачивать следующие налоги:
- Налог на прибыль (18%)
- НДС (20%)
- Подоходный налог с работников
- Социальные отчисления

Для малых предприятий предусмотрены льготы по налогам.
```

### Ответы на вопросы договоров
```
Для получения лицензии необходимо:
1. Подать заявление в уполномоченный орган
2. Представить пакет документов
3. Оплатить государственную пошлину
4. Пройти проверку соответствия требованиям

Срок рассмотрения заявления составляет 30 календарных дней.
```

## Тестовые сценарии

### Сценарий 1: Создание и публикация вопроса
1. Создать вопрос: "Как зарегистрировать ООО в Беларуси?"
2. Найти вопрос в панели модерации
3. Опубликовать вопрос
4. Проверить что вопрос отображается на сайте

### Сценарий 2: Ответ на вопрос
1. Создать вопрос: "Какие налоги платит ИП?"
2. Найти вопрос в панели модерации
3. Ответить на вопрос: "ИП платит налог на прибыль 18%"
4. Проверить что ответ опубликован

### Сценарий 3: Комплексная модерация
1. Создать несколько вопросов
2. Ответить на них с разными типами ответов
3. Проверить статусы всех вопросов
4. Убедиться в корректности публикации

## Рекомендации по использованию

### Генерация тестовых данных
Используйте `QuestionFactory` для генерации уникальных вопросов:
```python
from framework.utils.question_factory import QuestionFactory

factory = QuestionFactory()
question = factory.generate_question(category="регистрация")
```

### Проверка результатов
- Всегда проверяйте статусы вопросов и ответов
- Убедитесь в корректности опубликованного контента
- Проверяйте отображение в панели модерации
- Тестируйте различные типы ответов

## Частые ошибки

### Ошибки при создании вопросов
- Пустой текст вопроса
- Слишком длинный текст вопроса
- Некорректная категория

### Ошибки при ответах
- Пустой текст ответа
- Ответ на несуществующий вопрос
- Некорректный тип ответа

### Ошибки публикации
- Публикация уже опубликованного вопроса
- Публикация вопроса с неправильным статусом
