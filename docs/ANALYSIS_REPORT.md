### 4. Критерии проверки ссылок

Для каждой ссылки проверяем:
1. **Доступность** - HTTP 200 OK
2. **Переход** - корректный переход по ссылке  
3. **Целевая страница** - наличие заголовка, соответствующего тексту ссылки
4. **Контент** - базовая проверка содержимого страницы
