# Проект автотестов с авторизацией через куки

- Все чувствительные данные хранятся в папке `data` и не коммитятся в git.
- Для запуска тестов установите зависимости: `pip install -r requirements.txt`
- Для первой авторизации заполните `data/creds.env` и выполните `python src/auth_manager.py`.
- После этого автотесты будут использовать куки из `data/cookies.json`.
