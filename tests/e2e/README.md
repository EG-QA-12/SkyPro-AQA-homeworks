# Папка e2e (инфраструктурные файлы)

В этой директории больше не содержатся e2e-тесты. Все ключевые тесты перенесены в папку `integration`.

Папка используется для хранения инфраструктурных файлов, заглушек и настроек, необходимых для поддержки совместимости и исторических целей.

Если требуется добавить новый e2e-тест, обсудите это с командой и уточните, не должен ли он быть интеграционным.
