# SSO-тесты (Single Sign-On)

В этой директории находятся интеграционные тесты для проверки SSO-авторизации на различных доменах BLL.

## Основные файлы
- `test_sso_cookie_auth.py` — основной набор тестов SSO
- `constants.py` — константы и настройки для тестов
- `conftest.py` — фикстуры и вспомогательные функции

## Запуск тестов
```bash
pytest tests/integration/sso/ -v
```

Тесты были перенесены из e2e и теперь поддерживаются как часть интеграционного тестирования.
