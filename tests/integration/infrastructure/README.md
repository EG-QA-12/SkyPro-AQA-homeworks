# Инфраструктурные интеграционные тесты

В этой директории размещаются вспомогательные и инфраструктурные тесты, которые не относятся напрямую к бизнес-логике, но обеспечивают стабильность и корректность работы интеграционного слоя.

## Примеры тестов
- Проверка доступности сервисов
- Проверка базовых сценариев взаимодействия между компонентами

## Запуск
```bash
pytest tests/integration/infrastructure/ -v
```

Если требуется добавить новый инфраструктурный тест — создайте отдельный файл и опишите его назначение в docstring.
